# Generate a JWT token

# Install
```bash
yarn install
```

# Modify
## Privacy Key
```bash
privateKey = 'YOUR-PRIVACY-KEY'
``` 

## kid String
You can get it from the Line Developper channel dashboard.
```bash
kid: "YOUR-KID-STRING"
```

# Run
```bash
yarn start
```
